## 🚀 Getting Started

1. Clone this repository:
```bash
git clone https://github.com/AliAskariGithub/python-practice-projects.git
```

2. Navigate to project directory:
```bash 
cd python-practice-projects
```

3. Run any project:
```bash
python projectname.py
```
